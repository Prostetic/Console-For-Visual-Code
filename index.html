How to use Console

After downloading Visual code, start making a folder to code on in files. Next, open Visual Code and and press on the option the top left corner of file and in the dropdown, select open folder and then select the folder you named, then add a new file to a folder and name the three compilers index.html, style.css, main.js. The setup is now complete.

The console screen is operating system window displaying written commands from code from the compiler format, to start editing on the console screen go to main.js and write "console.log()" and then in teh console you can define varibles using var and turn them into a output using console.log() and the varible name inside the brackets.
